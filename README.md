# Bike-Helmet-Detection
Helmet & License Plate Detector (YOLOv8 + EasyOCR)

This project detects helmets and license plates using a custom-trained YOLOv8 model, and then performs OCR on the detected license plates using EasyOCR.

Features
  YOLOv8 object detection (Helmet, License Plate)
  OCR using EasyOCR for license plate text extraction
  Saves annotated output images
  Training script included
  Works on CPU or GPU
